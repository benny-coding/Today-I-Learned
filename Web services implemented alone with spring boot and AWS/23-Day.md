## AWS에 데이터베이스 환경을 만들어보자 - AWS RDS
  ### RDS 인스턴스 생성하기
    - MariaDB는 가격이 저렴하다
    - Amazon Aurora(오로라) 교체에 용이성이 있다.
      - AWS에서 __MySQL과 PostgreSQL을 클라우드 기반에 맞게 재구성한 데이터베이스__ 입니다.
    - MariaDB은 MySQL 기반으로 만들어졌으며 다음과 같은 장점이 있다.
      - 동일 하드웨어 사양으로 MySQL보다 향산된 성능
      - 좀 더 활성화된 커뮤니티
      - 다양한 기능
      - 다양한 스토리지 엔진
